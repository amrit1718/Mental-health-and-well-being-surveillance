# Mental Health and Well-being Surveillance, Assessment and Tracking System for Children

This project is a full-stack prototype built with:

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js with Express
- Database: MongoDB-ready schema with an in-memory demo fallback

## Run

1. Install packages:
   `npm install`
2. Copy `.env.example` to `.env`
3. Optional: add your MongoDB connection string to `MONGO_URI`
4. Start the server:
   `npm start`
5. Open:
   `http://localhost:5000`

If `MONGO_URI` is not provided, the app runs in demo mode using in-memory storage.
